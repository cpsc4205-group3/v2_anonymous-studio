# Retrospective Screenshots

Place sprint project-board screenshots in this directory.

**Naming convention:** `sprint<N>-<description>.png`

Examples:
- `sprint2-board-view.png`
- `sprint2-table-view.png`
- `sprint3-board-view.png`

Screenshots are referenced from `docs/retrospectives/SPRINTN-RETROSPECTIVE.md` files using relative paths:

```markdown
![Sprint N Board View](screenshots/sprintN-board-view.png)
```
